# user-agent strings

programatically: https://feederbox826.github.io/user-agents/user-agents.json

Latest user-agent strings for major browsers - same format as [jnrbsn/user-agents](https://github.com/jnrbsn/user-agents)

- Chrome [laststable, stable] on [mac, windows, linux]
- Firefox [stable, esr] on [mac, windows, linux, ubuntu]
- Safari stable on OSX
- Edge Stable on Windows

Gets user-agents directly from the source (except safari through [mdn/browser-compat-data](https://github.com/mdn/browser-compat-data))
